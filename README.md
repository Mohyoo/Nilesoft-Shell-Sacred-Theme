## Nilesoft Shell - The Sacred Theme
A beautiful theme for a beautiful shell
![Screenshot](preview.png)

## Intallation
Just download 'theme.nss' file and place it at the 'imports' folder inside your Nilesfot Shell's installation directory.

## Disclamer
I didn't hardcode this 'theme.nss' file on my own, and I'm not an expert in the nilesoft shell syntax; I only followed this [tutorial]
(https://www.youtube.com/watch?v=HwDcOhlaJsQ), started messing with the values like a kid, and voila! I got this.
